# Usage

## Basic Command
```bash
goto <pattern>
```

### Example
```bash
goto src
```

----

## Options

| Option | Description |
| ----------- | ------------------------------------- |
| `-d` | Specify maximum search depth. |
| `<pattern>` | Fuzzy search pattern for directories. |
